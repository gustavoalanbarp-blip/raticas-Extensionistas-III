TÍTULO DO PROJETO (Sistema de Agendamento Clínico)
Autor: Gustavo Alan Barp 

Introdução: O presente projeto propõe o desenvolvimento de um sistema de gerenciamento e agendamento de consultas para clínicas médicas. Com a digitalização dos serviços de saúde, torna-se essencial ferramentas que otimizem o tempo de médicos e pacientes.
Objetivo: O estudo tem por objetivo modelar a estrutura de dados e as interações de um sistema que permita a marcação de consultas por pacientes e o gerenciamento de agendas por parte dos médicos.
Metodologia: Trata-se de uma pesquisa aplicada focada em engenharia de software, utilizando a linguagem UML (Unified Modeling Language) para o mapeamento de Casos de Uso, Classes, Atividades  Sequência, além da ferramenta brModelo para a criação dos modelos Conceitual e Lógico de banco de dados.
Resultados: Identifica-se a modelagem completa dos requisitos do sistema, detalhando as entidades principais como Paciente, Médico e Consulta. O fluxo de decisão para agendamentos foi estruturado, garantindo o tratamento de indisponibilidade de horários.
Conclusão: Conclui-se que a modelagem estruturada antes do desenvolvimento do código (pasta app) garante um sistema mais robusto e livre de erros de lógica, facilitando a visualização da jornada do paciente ao agendar uma consulta.

Palavras-chave: Agendamento Médico. Modelagem de Dados. UML. Engenharia de Software.
